About Me
Opps! I found bugs in my code.

Earlier when i was not so familiar with programming i used to find people talking about bugs in their programs and i used to wonder are these real bugs that have eight legs and multiple eyes and spread various diseases? Well turns out they actually do! But not like what i described them as but you can say they are similar to their characteristics. Say where do you find bugs? Somewhere that isn't so clean and most bugs love that isn't it? This is what I learnt about bugs in programming! When you don't write a clean code you always end up getting small bugs which make the code look very dirty and unacceptable so during my journey in becoming a Software Developer i concentrated more on writing a clean code rather than just writing the code for the sake of getting the right results. I came to a conclusion that my first step in being a good programmer is to write "Clean and Understandable code" for anyone who has an idea about coding. This is the most important thing. I will never stop learning day by day because each day you have a chance to improve and I would never miss a chance to improve myself.
